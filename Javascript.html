Javascript

Javascript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

Javascript Programms

JavaScript is a powerful and flexible programming language. It can execute on a web browser that allows us to make interactive webpages such as popup menus, animations, form validation, etc.

Javascript Statement

JavaScript Statements. JavaScript statements are the commands to tell the browser to what action to perform. Statements are separated by semicolon (;). JavaScript statement constitutes the JavaScript code which is translated by the browser line by line.

JavaScript Keywords

Keywords are reserved words that are part of the syntax in the programming language. For example, const a = 'hello'; Here, const is a keyword that denotes that a is a constant.

console.log

The console. log() is a function in JavaScript that is used to print any kind of variables defined before in it or to just print any message that needs to be displayed to the user. Syntax: console. log("");

JavaScript allows you to work with three primitive data types:

  • string
  • number
  • boolean
  • biglnt
  • Undefined
  • null
  • object
  • symbol

String

A string is a sequence of one or more characters that may consist of letters, numbers, or symbols.

Number

The JavaScript Number type is a double-precision 64-bit binary format IEEE 754 value, like double in Java or C#. This means it can represent fractional values

Boolean

In JavaScript, a boolean value is one that can either be TRUE or FALSE. If you need to know “yes” or “no” about something, then you would want to use the boolean function.

Biglnt

JavaScript BigInt variables are used to store big integer values that are too big to be represented by a normal JavaScript Number .

Undefined

A variable that has not been assigned a value is of type undefined . A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value.

Null

In JavaScript, null is a special value that represents an empty or unknown value. For example, let number = null; The code above suggests that the number variable is empty at the moment and may have a value later.

Object

In JavaScript, an object is a standalone entity, with properties and type. Compare it with a cup, for example. A cup is an object, with properties. A cup has a color, a design, weight, a material it is made of, etc.

Symbol

Symbols are often used to add unique property keys to an object that won't collide with keys any other code might add to the object, and which are hidden from any mechanisms other code will typically use to access the object.

Identifier

Identifiers are JavaScript names. Identifiers are used to name variables and keywords, and functions. Variable: to store data - const/let/var Variable-name - Variable-name = value
Variables are Containers for Storing Data variable declaration.

There are three types of keywords for a variable declaration in JS: var, let, and const. var x; const y; let z; Var - It is an outdated keyword in Javascript. We cannot declare a variable without var in older browsers.

Var

var is the keyword that tells JavaScript you're declaring a variable. x is the name of that variable. = is the operator that tells JavaScript a value is coming up next. 100 is the value for the variable to store.

Let

The keyword let in Javascript is used to declare a variable that is block scoped in JavaScript. In JavaScript, the var keyword is typically used to declare a variable that is treated like any other variable, whereas variables declared using the let in JavaScript are block scoped.

const

Const is another keyword to declare a variable when you do not want to change the value of that variable for the whole program.

JavaScript Expressions

An expression is a combination of values, variables, and operators, which computes to a value. The computation is called an evaluation. The values can be of various types, such as numbers and strings.

Javascript Operators

Operators are used to performing specific mathematical and logical computations on operands.
